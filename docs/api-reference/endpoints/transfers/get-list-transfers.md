# GET /transfers — List transfers

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `GET`
- **Path:** `/transfers`
- **Operation ID:** `listTransfer`
- **Tags:** `Transfers`
- **Required bearer scopes:** `payout:transfer:read`

## Description

Returns a paginated list of fund transfers, filtered by origin or destination account, with optional sorting and date filtering.

Required permissions:
 - `payout:transfer:read`

## Parameters

| Name | In | Required | Type | Description | Example |
|---|---|---:|---|---|---|
| `after` | query | no | `string \| null` | Returns the elements in the list that come after the specified cursor. |  |
| `before` | query | no | `string \| null` | Returns the elements in the list that come before the specified cursor. |  |
| `first` | query | no | `integer \| null` | Returns the first _n_ elements from the list. | 42 |
| `last` | query | no | `integer \| null` | Returns the last _n_ elements from the list. | 42 |
| `origin_id` | query | no | `string \| null` | Filter to transfers sent from this account. Accepts a user, company, or ledger account ID. |  |
| `destination_id` | query | no | `string \| null` | Filter to transfers received by this account. Accepts a user, company, or ledger account ID. |  |
| `order` | query | no | `CreditTransactionTransfersSortableColumns \| null` | The column to sort transfers by. Defaults to creation date. |  |
| `direction` | query | no | `Direction \| null` | The sort direction for results. Defaults to descending. |  |
| `created_before` | query | no | `string \| null / date-time` | Only return transfers created before this timestamp. | 2023-12-01T05:00:00.401Z |
| `created_after` | query | no | `string \| null / date-time` | Only return transfers created after this timestamp. | 2023-12-01T05:00:00.401Z |
