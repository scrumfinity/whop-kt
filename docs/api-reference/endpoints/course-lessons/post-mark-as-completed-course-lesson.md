# POST /course_lessons/{lesson_id}/mark_as_completed — Mark as completed course lesson

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `POST`
- **Path:** `/course_lessons/{lesson_id}/mark_as_completed`
- **Operation ID:** `markAsCompletedCourseLesson`
- **Tags:** `Course lessons`
- **Required bearer scopes:** _None documented_

## Description

Mark a lesson as completed for the current user after they finish the content.

## Parameters

| Name | In | Required | Type | Description | Example |
|---|---|---:|---|---|---|
| `lesson_id` | path | yes | `string` | The unique identifier of the lesson to mark as completed (e.g., "lesn_xxxxxxxxxxxxx"). | lesn_xxxxxxxxxxxxx |
