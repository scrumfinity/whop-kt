# GET /authorized_users — List authorized users

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `GET`
- **Path:** `/authorized_users`
- **Operation ID:** `listAuthorizedUser`
- **Tags:** `Authorized users`
- **Required bearer scopes:** `company:authorized_user:read`, `member:email:read`

## Description

Returns a paginated list of authorized team members for a company, with optional filtering by user, role, and creation date.

Required permissions:
 - `company:authorized_user:read`
 - `member:email:read`
