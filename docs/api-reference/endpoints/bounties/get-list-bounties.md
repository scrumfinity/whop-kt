# GET /bounties — List bounties

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `GET`
- **Path:** `/bounties`
- **Operation ID:** `listBounty`
- **Tags:** `Bounties`
- **Required bearer scopes:** _None documented_

## Description

Returns a paginated list of workforce bounties. When experienceId is provided, returns bounties scoped to that experience. When omitted, returns bounties with no experience.

## Parameters

| Name | In | Required | Type | Description | Example |
|---|---|---:|---|---|---|
| `after` | query | no | `string \| null` | Returns the elements in the list that come after the specified cursor. |  |
| `before` | query | no | `string \| null` | Returns the elements in the list that come before the specified cursor. |  |
| `first` | query | no | `integer \| null` | Returns the first _n_ elements from the list. | 42 |
| `last` | query | no | `integer \| null` | Returns the last _n_ elements from the list. | 42 |
| `experience_id` | query | no | `string \| null` | The experience to list bounties for. When omitted, returns bounties with no experience. | exp_xxxxxxxxxxxxxx |
| `status` | query | no | `Statuses \| null` | Filter bounties by status. |  |
| `direction` | query | no | `Direction \| null` | Sort direction. Defaults to descending. |  |
