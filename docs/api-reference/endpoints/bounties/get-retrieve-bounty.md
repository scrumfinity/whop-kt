# GET /bounties/{id} — Retrieve bounty

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `GET`
- **Path:** `/bounties/{id}`
- **Operation ID:** `retrieveBounty`
- **Tags:** `Bounties`
- **Required bearer scopes:** _None documented_

## Description

Retrieves a workforce bounty for the current authenticated user.

## Parameters

| Name | In | Required | Type | Description | Example |
|---|---|---:|---|---|---|
| `id` | path | yes | `string` | The unique identifier of the workforce bounty to retrieve. | bnty_xxxxxxxxxxxxx |
