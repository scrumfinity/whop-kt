# GET /users/{id}/access/{resource_id} — Check access

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `GET`
- **Path:** `/users/{id}/access/{resource_id}`
- **Operation ID:** `checkAccess`
- **Tags:** `Users`
- **Required bearer scopes:** _None documented_

## Description

Check whether a user has access to a specific resource, and return their access level.

## Parameters

| Name | In | Required | Type | Description | Example |
|---|---|---:|---|---|---|
| `id` | path | yes | `string` | The unique identifier or username of the user. | user_xxxxxxxxxxxxx |
| `resource_id` | path | yes | `string` | The unique identifier of the resource to check access for. Accepts a company, product, or experience identifier. | prod_xxxxxxxxxxxxx |

## Request body

_No request body._
