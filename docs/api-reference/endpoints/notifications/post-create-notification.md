# POST /notifications — Create notification

Generated from Whop's official documented OpenAPI source: [openapi.documented.yml](../../openapi.documented.yml).

- **Base URL:** `https://api.whop.com/api/v1`
- **Method:** `POST`
- **Path:** `/notifications`
- **Operation ID:** `createNotification`
- **Tags:** `Notifications`
- **Required bearer scopes:** `notification:create`

## Description

Send a push notification to users in an experience or company team. The notification is processed asynchronously and supports targeting specific users.

Required permissions:
 - `notification:create`

## Parameters

_No path, query, header, or cookie parameters._

Required: **yes**
