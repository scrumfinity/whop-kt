# Changelog

All notable changes to `whop-kt` will be documented in this file.

## [Unreleased]

### Added

- Kotlin/JVM SDK foundation with Ktor transport and kotlinx.serialization.
- Typed resources for checkout configurations, payments, products, plans, and users.
- Cursor pagination models and auto-paging flows.
- Structured Whop API error model and exceptions.
- Standard Webhooks verifier and typed common event wrappers.
- OAuth 2.1 + PKCE helpers.
- Iframe user-token verifier with caller-supplied public key provider.
- KDoc for public SDK methods and models.
- MockEngine-based unit tests.
